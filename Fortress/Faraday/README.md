## Faraday – Conceptual Notes

**Topics Learned:**
- Web application enumeration
- API endpoint discovery and analysis
- Authentication bypass techniques

---

**Key Learning Points:**
- Hidden API endpoints can expose sensitive functionality if not properly secured
- Weak authentication mechanisms can lead to unauthorized access
- Extracted credentials often require offline cracking to be useful
- Proper enumeration is critical for identifying attack paths
- Chaining vulnerabilities is essential for full system compromise
- Kernel-level rootkits can hide system activity by hooking critical functions
