## Akerva Fortress – Conceptual Notes

![Akerva Fortress](images/pwned-akerva.png)

---

### **Topics Learned:**

* SNMP enumeration and information gathering
* Backup file discovery and analysis
* Flask debug mode exploitation
* Local File Inclusion (LFI) concepts
* Reverse shell execution
* Linux privilege escalation techniques
* Cipher decoding (Base64 + Vigenère)

---

### **Key Learning Points:**

* SNMP services can leak valuable internal information if misconfigured
* Backup files often expose source code, credentials, and hidden endpoints
* Flask debug mode can lead to full application compromise if left enabled
* LFI vulnerabilities allow attackers to read sensitive files on the system
* Reverse shells are essential for gaining interactive access after exploitation
* Privilege escalation is required to fully compromise the system
* Combining multiple vulnerabilities is key to completing complex attack chains
* Basic cryptographic techniques help in decoding hidden or encoded data

---

### **Skills Strengthened:**

* Network enumeration (SNMP, service discovery)
* Backup file fuzzing and analysis
* Web vulnerability identification (LFI, debug exploitation)
* Reverse shell handling and stabilization
* Linux post-exploitation enumeration
* Privilege escalation techniques
* Cryptography basics (encoding & classical ciphers)
